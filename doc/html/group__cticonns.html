CTI connections from ARM TRM.

Lists of CTI connections from ARM TRM material.

Indicate that a CTI in/out port is connected to some other component (e.g. ETB, STM). These connections will have been decided as part of the SoC design.

The "trigger port id" on the other component is an index defined for convenience by this library - its value does not correspond to anything in the CoreSight architecture.

Connections from the CPU to its CTI are defined in each CPU's TRM.
