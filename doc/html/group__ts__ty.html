enum cs_ts_gen_interface_t

TS generator access type.

timestamp generators have two possible access interfaces, one read-only and one read-write for configuration and control.

Generators in a CoreSight system must implement the read-write interface, but this API can be connected to a system generator that has the read-only access.

Enumerator
TSGEN_INTERFACE_CTRL 

interface is the control interface - used for both CS and secure control of system generic time.

TSGEN_INTERFACE_RO 

interface is the read-only access interface - used if generator using system generic time.
