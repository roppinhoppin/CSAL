Sample the program counter on a running CPU.

Routine will detect architecture v7 or architecture v8 core and sample appropriately. If the library is built for 64bit virtual addresses (cs_virtaddr_t) then a 64 bit address will be returned if a v8 64 bit core is being sampled.

Optionally, CONTEXTID and VMID can be sampled, synchronously with the PC.

Parameters
devdevice to sample - must be the debug registers on a core.
pcpointer to receive PC virtual address sample. Bit 0 set indicates Thumb state.
cidpointer to receive CONTEXTID sample
vmidpointer to receive VMID sample
Returns
0 if valid sample was obtained. -1 if sampling is not possible at present time.
