◆ cs_etm_clean()

int cs_etm_clean ( cs_device_t  dev)

Put an ETM into a "clean" state.

This is not necessarily a "reset" state as an ETM reset does not initialize the registers. On exit from cs_etm_clean(): Counters will be zeroed and incrementing NEVER. Sequencer will be in state 0 with all transitions NEVER. External outputs will all be NEVER. Trace ID will be retained at the current value.
