Detailed Description

This API group tells the library about how the trigger inputs and outputs on non-CTI components (ETMs, ETBs etc.), are connected to CTI ports in the cross-trigger fabric.

It is not necessary to register the connections between CPU triggers (e.g. EDBGRQ) and the CPU's own CTI.
