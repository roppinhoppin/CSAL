STM config operation flags.

Bitflags defining the register blocks to read or write in dynamic config structure.

 clear the flags
 
#define CS_STMC_CTRL   0x0001

#define CS_STMC_SYNC   0x0002
 SYNCR.
 
#define CS_STMC_PENA   0x0004
 Port enable regs (SPER, SPTER, SPSCR, SPMCR, PRIVMASKR.
 
#define CS_STMC_OVER   0x0008
 Override regs (OVERIDERR, MOVERRIDER)

 Trigger control (SPTRIGCSR)
